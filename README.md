A simple React application with a Login page and a Dashboard page. The project demonstrates form validation, React hooks, and dynamic UI updates.



**Features**



&nbsp;	Login using email or username



&nbsp;	Password validation (uppercase, lowercase, number, special character, 6+ characters)



&nbsp;	Modern, attractive UI



&nbsp;	Dashboard displays:

&nbsp;		Welcome message

&nbsp;		Logged-in user

&nbsp;		Login history stored using localStorage

&nbsp;		Logout button with profile icon



**Technologies Used**



* React
* React Router
* useState \& useEffect hooks
* CSS (custom styles)
* LocalStorage API



**Installation**



Install project dependencies:



&nbsp;	npm install





Start the development server:



&nbsp;	npm start
